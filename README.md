# Image Coordinate Detection Tool

PixelMap is an interactive web application that allows users to upload an image and extract pixel coordinates (x, y) by clicking or entering values. It is useful for image mapping, UI design, and basic computer vision tasks.

---

## 🚀 Features

* 📤 Upload images (JPG, PNG)
* 🎯 Extract pixel coordinates (x, y)
* 🔲 Supports rectangle and circle coordinate logic
* ⚡ Real-time coordinate display
* 🌐 Deployable using Streamlit Cloud

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend / Deployment:** Python, Streamlit
* **Libraries:** Pillow

## ⚙️ Installation & Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/pixelmap-image-coordinates.git
cd pixelmap-image-coordinates
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
[streamlit run app.py](https://image-coordinate-finder-gna5qerghyuai9b3mwgdli.streamlit.app/)
```

---

## ☁️ Deployment (Streamlit Cloud)

1. Push code to GitHub
2. Go to Streamlit Cloud
3. Select your repository
4. Choose `app.py`
5. Click Deploy
